# 🌿 Paradise Nursery

## 📌 Descripción del Proyecto
Paradise Nursery es una aplicación web de comercio electrónico desarrollada con React.  
Permite a los usuarios explorar una variedad de plantas de interior, ver sus detalles (imagen, nombre, descripción y precio), agregarlas a un carrito de compras y gestionar los productos seleccionados.

---

## 🎯 Objetivo
El objetivo de este proyecto es aplicar los conocimientos adquiridos en React para construir una aplicación dinámica e interactiva, simulando una tienda en línea funcional y fácil de usar.

---

## 🚀 Funcionalidades principales

- 🌱 Visualización de plantas organizadas por categorías
- 🛒 Agregar productos al carrito de compras
- 🔢 Contador dinámico de productos en el carrito
- ➕➖ Incrementar y disminuir cantidades de productos
- ❌ Eliminar productos del carrito
- 💰 Cálculo automático del costo total
- 🧭 Barra de navegación entre páginas (Inicio, Plantas, Carrito)
- 🖼️ Página de inicio con diseño visual atractivo

---

## 🛠️ Tecnologías utilizadas

- React.js
- JavaScript (ES6)
- CSS
- Redux Toolkit (para manejo del estado)
