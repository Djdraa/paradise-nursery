# 🌿 PARADISE NURSERY (e-plantShopping)

Aplicación web de tienda de plantas con carrito de compras.

## 🚀 Funcionalidades

* Listado de plantas por categorías
* Carrito de compras dinámico
* Agregar, eliminar y actualizar cantidades de productos
* Navegación entre páginas

## 🛠 Tecnologías

* React
* Redux Toolkit
* React Router

## 📌 Descripción

**e-plantShopping (Paradise Nursery)** es una aplicación web que permite a los usuarios explorar una variedad de plantas de interior, agregarlas al carrito de compras y gestionar sus pedidos de forma sencilla e interactiva.
